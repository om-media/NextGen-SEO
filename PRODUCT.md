# Product

## Register

product

## Users

SEO practitioners and teams who need to review Search Console, GA4, Bing, crawl, ranking, and referral data in one workspace without exporting reports into spreadsheets first.

## Product Purpose

NextGen SEO is an open-source, warehouse-first SEO analysis workspace. It imports all supported history after a data source is connected, keeps that history current in the background, and serves repeatable reports from stored data. Success means users can switch sites, sources, properties, dates, and analytical views without stale selections, hidden live fetches, or long blocking imports.

## Brand Personality

Focused, credible, and calm. The interface should feel like a dependable professional tool that makes complex source data easier to inspect without marketing noise.

## Anti-references

- SaaS upgrade prompts, billing assumptions, artificial feature gates, or plan language.
- Decorative dashboard composition that competes with the data.
- Ambiguous loading, silent source changes, stale values, and controls that trigger unexpected imports.
- Empty states that blame the user or expose implementation details such as missing warehouse tables.

## Design Principles

1. Keep source identity stable: a user's selected site and property remain selected until the user changes them or the source proves they are unavailable.
2. Read from the warehouse: navigation and report filters never depend on hidden live provider requests.
3. Keep background work quiet: existing data stays usable while imports run, and progress appears only when it materially affects the current view.
4. Prefer complete analytical workflows over export-first workflows.
5. Make system state precise: distinguish no data, loading, disconnected access, and failed imports without conflating them.

## Accessibility & Inclusion

Target WCAG 2.1 AA. Preserve keyboard navigation, visible focus states, semantic controls, sufficient contrast, reduced-motion behavior, and status text that does not rely on color alone.
